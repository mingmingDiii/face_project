Dong M, Wen S, Zeng Z, et, al.

Neurocomputing

内容： 构建了针对人脸分割场景下的稀疏网络，同时达到了 LFW 分割最高精度

Wen S, Dong M, Yang Y, et al.（导师一作）

IEEE TNNLS （在审）
